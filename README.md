## サービス概要

「ひだまり日記」 は、日常の幸せや、感謝、ちょっと頑張った自分を気軽に記録できるアプリです。

「今日掃除した、えらい！」「お気に入りの音楽でリラックスできた。ありがとう」――そんな一言も、日々の幸せとして残せます。  

続けていくうちに、はじめは何も思わなかった出来事にも「感謝」や「幸せ」に気づけるようになり、  
**気持ちが前向きになったり、自己肯定感の向上、ストレスが和らぐ効果があります。**  

記録した日記は、非公開にして自分だけのために残すことも、公開してみんなと分かち合うこともできます。

## 💭開発背景

開発のきっかけは、私自身がネガティブな思考に苦しんだ経験です。  
嫌なことがあった日、そのことばかり考えてしまう。気分転換にと開いたSNSではネガティブな情報に触れ、さらに落ち込んでしまう。 また、一度落ち込むと、普段の何気ない物事までネガティブに捉えてしまうということがありました。  
そんな時、「幸せなことだけが書かれているSNSがあったらいいのに」と思ったことがきっかけです。

ネガティブな状態から抜け出したいと「ポジティブになる方法」を調べる中で、私は「感謝日記」に出会いました。  
これは、「毎日3つほど感謝した出来事を書き出す」という、とてもシンプルな習慣です。

この「感謝」がもたらす効果は非常に大きく、ポジティブになれるだけでなく、「自己肯定感の向上」「モチベーションの維持」「身体的な健康」など、多岐にわたるメリットがあると知り、アプリに取り入れたいと考えました。

「感謝日記」 「幸せなことだけを共有できるSNS」

この2つのアイデアが結びつき、感謝日記を書き、それを温かいコミュニティで共有できるアプリ『ひだまり日記』を開発するに至りました。

## 🎯ターゲット層

- 自己肯定感を少しずつ高めたい方
- ネガティブな気持ちに引っ張られず、日常を前向きに過ごしたい方
- SNSで流れてくるネガティブな情報に触れて、気持ちが沈んでしまう方  
- いいねやフォロワー数など、他人との比較に疲れてしまった方  
- 日々忙しく、自分を振り返る時間がなかなか取れない方
  
## 🌼ひだまり日記の使い方イメージ

- **今日の幸せ・感謝・頑張ったことを記録する**  
例：「美味しいコーヒーを飲んだ、ありがとう」「掃除をして部屋がきれいになった、えらい！」  
(記録した「幸せ」「感謝」「頑張り」は、丸やハート、クローバーのかわいい形になって、アプリ内の「幸せの瓶」に少しずつ貯まっていきます。)

- **みんなの幸せに触れる**  
「みんなの日記」ページでは、他のユーザーが共有してくれた感謝日記を読むことができます。  
ここにはポジティブな出来事だけが流れるので、安心して他の人の幸せに触れられます。読むだけで、心がぽかぽかと温かくなるはずです。

- **自分の「幸せの記録」を読み返す**  
心が沈んだときや疲れたときは、過去の自分の日記や、幸せでいっぱいになった「瓶」を眺めてみてください。  
「自分はこんなに幸せを感じていたんだ」と再確認することが、ネガティブな気分に飲み込まれそうな時に、心を立て直すきっかけをくれます。

- **ポジティブな習慣が身につく**  
日記を続けるうちに、日常の中から「幸せ」を見つけるのがだんだん上手になります。 「ひだまり日記」は、そんなポジティブな心の習慣が自然と身につく場所です。


## 🌟サービス差別化のポイント・推しポイント

| サービス | 特徴 | 差別化ポイント |
|----------|------|----------------|
| シンプル日記 | 何でも記入可能。通常の日記をそのままアプリ化 | ✅ **ひだまり日記**は「幸せ・感謝・自分を褒めること」に特化。意識的に前向きになれる習慣作りが目的。単なる日記以上の価値を提供。 |
| 感謝の庭 | 感謝をテーマ、瞑想あり、共有機能なし。マイページには「感謝数・日記数」の表示しかなく、モチベーション維持がしづらい | ✅ **ひだまり日記**は書いた内容を幸せを瓶に貯めて可視化する機能により、書いた内容が目に見える形で積み重なり、モチベーションが維持しやすい。 |
| マイ日記 | テンプレート豊富（感謝・感情・恋愛など）、共有機能なし | ✅ **ひだまり日記**は日記の共有機能があり、あたたかいコミュニティ空間を提供 |

### 継続しやすいポイント 🌟

- **幸せが貯まる「幸せ瓶」** 🍀  
記録した幸せが「幸せ瓶」に目に見える形で貯まっていきます。 「こんなに幸せが積み重なったんだ」という達成感が、日々のモチベーションに。 瓶をいっぱいにすると幸せコレクターレベルがアップ！バッジがもらえる楽しみもあります！

- **日記に「AIコメント」** 🤖  
投稿した日記に、AIが優しいコメントを返してくれます。 あなたの幸せや頑張りに共感し、ポジティブな言葉をかけてくれるので、「読んでもらえた！」という嬉しさが「また明日も書こう」というモチベーションにつながります。

- **自動作成される「月間の振り返り」** 🗓️  
その月にどんな幸せがあったか、アプリが自動で「振り返り」を作成します。 「私、こんなに幸せを感じていたんだ」「こんな頑張りがあったな」と再確認することが、ポジティブ習慣につながります。

- **励みになる「みんなの日記」** 💓  
「みんなの日記」をのぞけば、他の人の温かい投稿にふれられます。

- **忘れない「リマインド通知**」 ⏰  
毎日設定した時間に通知が届くので、うっかり書き忘れるのを防げます。 日記を書く習慣づくりを、やさしくサポートします。

## 技術スタック

**言語・フレームワーク**

- Ruby 3.2
- Ruby on Rails 7.2.2

**フロントエンド**

- Stimulus + Stimulus Autocomplete（検索機能、ユーザー検索/キーワード/ハッシュタグ）
- matter.js（幸せを瓶に貯めて可視化するアニメーション）
- Chart.js（記録された幸せの数をグラフ化）

**通知・外部サービス**

- OneSignal（プッシュ通知、リマインド通知）
- OpenAI API（元気になる一言生成）

**ストレージ・デプロイ環境**

- Active Storage + Cloudflare R2
- Render
